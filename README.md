#  таймлайн

Этот проект реализует интерактивный таймлайн с вращающимся кругом, категориями и событиями, сверстанный по макету из Figma (1920×1080).  
При переключении категорий анимируются:

- вращение круга и активной точки;
- плавная смена названия категории;
- плавное появление/исчезновение событий таймлайна;
- кастомные стрелки навигации по кругу и по событиям.

## технологии

- React
- TypeScript (если у тебя TS, если нет — убери эту строку)
- SCSS / CSS-модули
- Swiper (для перелистывания событий)
- GSAP (для анимаций)

## требования

- Node.js (рекомендуется версия 18+)
- npm или yarn

## для установки и запуска в cmd прописываем следующее:

git clone https://github.com/danz222/timelinehistory.git
cd timelinehistory

## установить зависимости:

- npm install

## запустить проект и перейти по ссылке в браузер:

- npm start
